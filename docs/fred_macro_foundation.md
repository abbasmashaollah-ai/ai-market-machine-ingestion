# FRED Macro Foundation

This is the dry-run foundation for the FRED macro slice.

Boundary:
- no vendor calls by default
- no DB writes
- no AI/trading/risk/signal/regime logic

Starter series:
- DGS10
- DGS2
- FEDFUNDS
- CPIAUCSL
- UNRATE

The foundation uses deterministic fixture records until a live vendor and persistence contract are explicitly approved.
